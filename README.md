# 多多采集助手

拼多多商品信息抓取 Chrome 扩展，注入商品页面，模拟人工浏览采集商品数据并导出为 Excel--仅支持妙手ERP的格式（对应妙手的导入采集），然后就可以用妙手ERP上架到各个跨境电商平台了！

## 安装

1. Chrome → `chrome://extensions` → 开启"开发者模式"
2. "加载已解压的扩展程序" → 选择 `chrome-extension/` 目录
3. 打开拼多多移动端商品页即可使用

## 试用模式

首次安装可免费采集 **3 个商品**，无需激活码。试用用完需激活后才能继续使用。

## 功能

- 采集商品标题、主图、详情图
- 自动遍历 SKU 规格组合，获取每个 SKU 的价格和图片
- 模拟人工浏览行为（随机延迟、滚动、鼠标事件）
- 导出为妙手 ERP 兼容的 Excel 格式（XLSX / CSV）

## 激活

配套有激活服务器用于管理激活码的生成和验证。激活码格式为 36 位 UUID，输入后绑定设备。
